# PAGE_LAYOUT.md — RIFIM Group DDS v2.0
_(Berlaku untuk PT. RIFIM Internasional Gemilang, Menala, dan Lailan — 1 sistem desain, aset berbeda per perusahaan)_

## 1. Ukuran Halaman
- Format: **A4 Portrait** — 210mm x 297mm

## 2. Margin Halaman (resmi)
| Sisi | Ukuran |
|---|---|
| Atas | **1 cm** |
| Bawah | **1.5 cm** |
| Kiri | **2.5 cm** |
| Kanan | **2.5 cm** |

## 3. Margin Header & Footer (khusus, terpisah dari margin halaman)
- Header margin: **1 cm dari atas**
- Footer margin: **1 cm dari bawah**
- Ini HARUS disetel identik di pengaturan Header/Footer Google Docs
  (bukan cuma margin body biasa) — lihat [GOOGLE_DOCS.md](./GOOGLE_DOCS.md).

## 4. Dimensi Header & Footer
- Header: tinggi **38mm**, margin-top **10mm**, anchor ke page top,
  scale **fit width** (lihat [HEADER_SYSTEM.md](./HEADER_SYSTEM.md)).
- Footer: tinggi **32mm**, margin-bottom **10mm**, scale **fit width**
  (lihat [FOOTER_SYSTEM.md](./FOOTER_SYSTEM.md)).

## 5. Margin Body (isi surat)
- Kiri: 25mm
- Kanan: 25mm
- (Selaras dengan margin halaman kiri/kanan 2.5cm di atas.)

## 6. Aturan Page Break
- Header dan footer wajib muncul di setiap halaman.
- Tabel tidak boleh terpotong di tengah baris.
- Blok tanda tangan tidak boleh terpisah dari nama penandatangan.
