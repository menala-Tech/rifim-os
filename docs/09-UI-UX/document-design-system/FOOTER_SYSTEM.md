## 2. Dimensi & Penempatan
- Tinggi: **32mm**
- Margin bottom: **10mm**
- Scale: **Fit Width**, full-width, aspect ratio dipertahankan.

## 3. Aturan Wajib
- Ditempatkan di Footer section native (Google Docs/Word footer, bukan
  gambar manual di body).
- Background **transparan** (PNG alpha), tidak boleh ada kotak putih
  di baliknya.
- **Tidak boleh menimpa (overlap) isi dokumen** — kalau konten terlalu
  panjang, dorong ke halaman berikutnya, footer tidak boleh dikecilkan
  atau ditumpuk dengan teks.

## 4. Konten Dinamis di Atas Footer
Baris kecil nomor halaman (jika dokumen > 1 halaman) boleh ditambahkan
tepat di atas footer banner, TIDAK menimpa footer itu sendiri. Tidak ada
elemen dekoratif tambahan lain — footer sudah lengkap dari asetnya
sendiri.
