# apps/

> Frontend application RIFIM OS. Semua UI hanya berada di folder ini.

---

## Rules

- Jangan taruh business logic di sini — hanya UI layer
- Business logic berada di `modules/` dan `automation/`
- UI harus menggunakan komponen dari `branding/ui-kit/`

---

## Applications

| Folder | Type | Description |
|--------|------|-------------|
| `web/` | Web App | Aplikasi web utama (browser) |
| `mobile/` | Mobile Web | Tampilan mobile-optimized |
| `pwa/` | PWA | Progressive Web App (installable) |
