# PROMPT AI – TAMPILAN ROOM CHAT RIFIM CHAT
**Buat UI/UX room chat mobile aplikasi komunikasi operasional Maxim Airport dengan identitas RIFIM.**

---

## 1. HEADER ROOM CHAT
*   Tombol kembali (back)
*   Foto/Logo room di kiri
*   Nama room & jumlah peserta + online
*   Ikon telepon & video call grup
*   Menu opsi ( ⋮ ) di kanan

---

## 2. PESAN CHAT
*   Avatar di kiri setiap pesan
*   Nama, waktu, dan role (Admin) jika admin
*   Pesan teks, emoji reaction, file, gambar
*   Balon chat dengan sudut membulat
*   Warna pesan: admin (kuning muda), lainnya (hitam/abu gelap)

---

## 3. FITUR AKSI (MENU +)
Menyediakan akses cepat ke lampiran dengan ikon berikut: Foto, Kamera, File, Lokasi, Kontak, Polling.
*   Kirim foto dari galeri
*   Ambil foto langsung
*   Kirim file dokumen
*   Kirim lokasi
*   Kirim kontak
*   Buat polling/pertanyaan

---

## 4. MENU OPSI ( ⋮ )
*   **Info Room:** Info detail room & peserta
*   **Cari Pesan:** Cari pesan dalam room
*   **Bisukan Notifikasi:** Bisukan notifikasi room
*   **Media, File & Link:** Lihat media, file & link
*   **Pengaturan Room:** Pengaturan room (khusus admin)
*   **Keluar Room:** Keluar dari room

---

## 5. INDIKATOR & STATUS
*   **Online:** jumlah peserta online (contoh: `12 Online`)
*   **Admin:** label khusus admin (contoh: `Admin` warna kuning)
*   **Sistem/AI:** pesan sistem / peringatan AI

---

## 6. INPUT PESAN
Elemen antarmuka di bagian bawah layar:
*   Tombol `+` untuk fitur kirim
*   Input teks di tengah (`Ketik pesan...`)
*   Emoji picker
*   Tombol rekam suara

---

## 7. WARNA & TEMA
| Kode Warna | Keterangan |
| :--- | :--- |
| `#FFC700` | Kuning (Aksen utama / Identitas Maxim) |
| `#1E1E1E` | Hitam (Background utama) |
| `#2B2B2B` | Abu-abu gelap (Balon chat user) |
| `#FFFFFF` | Putih (Teks utama) |
| `#00C853` | Hijau (Indikator online / sukses) |
| `#FF5252` | Merah (Peringatan / Hapus) |

*   Tema gelap elegan dengan aksen kuning (identitas Maxim)
*   Kontras tinggi untuk keterbacaan
*   Balon admin: kuning muda
*   Balon user: abu gelap
*   Teks utama: putih / abu muda

---

## 8. TIPOGRAFI
*   **Font Family:** Poppins
*   **Judul Room:** SemiBold (16px)
*   **Nama Pengirim:** Medium (13px)
*   **Isi Pesan:** Regular (14px)
*   **Waktu:** Regular (11px)
*   **System/AI:** Medium (12px)
*   **Spasi antar pesan:** 8-12px

---

## 9. FITUR KHUSUS ROOM CHAT RIFIM
*   **Rekam Suara:** Rekam suara & kirim.
*   **Tag Peserta:** Tag nama peserta dengan `@nama`.
*   **Keluarkan Peserta (Admin):** Admin dapat mengeluarkan peserta dari room.
*   **Hapus Pesan (Admin):** Admin dapat hapus pesan.
*   **Silent Mode (Admin):** Mode di mana peserta tidak bisa kirim chat (Silent Mode).
*   **AI Guard:** Pesan diawasi oleh AI (hindari penggunaan kata kasar).
*   **Deteksi Link Mencurigakan:** Deteksi link mencurigakan & peringatan otomatis.

---

## 10. SPESIFIKASI TEKNIS
*   **Resolusi:** 1170 x 2532 px (iPhone 12/13/14)
*   **Safe Area:** 44px (atas & bawah)
*   **Radius Komponen:** 16px (bubble, button)
*   **Ikon Style:** Line & Filled (2px)
*   **Shadow:** Soft (Elevation 4)
*   **Mode Warna:** Dark Mode
*   **Bahasa:** Indonesia
