# PROMPT AI – TAMPILAN NOTIFIKASI RIFIM CHAT
**Buat UI/UX halaman notifikasi mobile aplikasi komunikasi operasional Maxim Airport dengan identitas RIFIM.**

---

## 1. HEADER
*   **Judul halaman:** "Notifikasi"
*   **Icon setting:** di kanan atas
*   **Warna header:** hitam
*   **Teks:** putih, bold

## 2. TAB FILTER
*   **Tab Menu:** `Semua` | `Belum Dibaca 8` | `Penting`
*   Tab filter untuk kategori notifikasi
*   "Semua" aktif (warna kuning)
*   "Belum Dibaca" menampilkan badge jumlah
*   "Penting" untuk notifikasi prioritas

## 3. DAFTAR NOTIFIKASI
Setiap item menampilkan: Icon kategori (berwarna), Judul notifikasi (bold), Deskripsi singkat, Waktu di kanan, Titik kuning jika belum dibaca. Urutan terbaru di atas.

*   **Driver Datang** (`08:31`): Andi - A023 sudah datang di counter 2
*   **Smart Queue** (`08:28`): Nomor antrian baru: B045
*   **Isi Saldo Berhasil** (`08:25`): Saldo Rp150.000 berhasil ditambahkan
*   **Approval Baru** (`08:20`): Permintaan approval: Invoice INV/2024/051
*   **Pengumuman** (`08:15`): Briefing pagi jam 07:00 di Meeting Point
*   **AI Insight** (`08:10`): Terdapat peningkatan driver idle di jam 13:00 - 15:00
*   **Warning** (`08:05`): Kata tidak pantas terdeteksi di Room Staff
*   **System** (`07:50`): Maintenance sistem malam ini 23:00 - 01:00
