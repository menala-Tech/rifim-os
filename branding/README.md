# branding/

> Seluruh identitas visual PT. RIFIM Internasional Gemilang.

---

## Rules

- Logo utama hanya boleh disimpan di folder ini — tidak boleh di folder lain
- Setiap dokumen perusahaan wajib menggunakan asset dari folder ini
- Jangan modifikasi logo utama tanpa persetujuan manajemen

---

## Corporate Colors

| Color | Hex | Usage |
|-------|-----|-------|
| RIFIM Red | `#CC0000` | Primary, header, accent |
| Dark Gray | `#2D2D2D` | Text, body |
| White | `#FFFFFF` | Background, clean space |
| Light Gray | `#F5F5F5` | Background secondary |

---

## Structure

| Folder | Isi |
|--------|-----|
| `logo/` | Logo utama (PNG, SVG, berbagai ukuran) |
| `icon/` | Icon & favicon |
| `font/` | Typography resmi perusahaan |
| `color-palette/` | Color palette & brand guidelines |
| `watermark/` | Watermark dokumen |
| `letterhead/` | Kop surat resmi |
| `document-style/` | Style guide dokumen |
| `ui-kit/` | Komponen UI perusahaan |
